예상 워크 플로우
1. (고객) 상표명, 상표 설명, 사진 입력
2. (시스템)
3. 키프리스 API로 유사 상표 검색
4. 검색된 상표들의 사진, 이름 등을 상표법, 특허청 심사 기준 등으로 평가
5. 유사도가 높아서 영향을 미칠 거 같은 것들은 그 상표의 의견제출통지서, 거절 통지서 내용을 보고 우리가 분석한 상표에도 해당하는지 분석
6. (결과) 예상 등록 확률, 유사 상표 목록(제 43류라면 구체적으로 어떤 카테고리인지 10~15개 정도(정확히는 유사군 코드), 카테고리 분류 ex) 43류 ~
