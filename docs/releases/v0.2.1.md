### 2. Favicon 최적화 및 현대화

#### Next.js 15 자동 생성 활용

```
/src/app/icon.png        # 512x512 메인 파비콘
/src/app/apple-icon.png  # 180x180 Apple touch icon
```

**Next.js의 자동 처리 기능**:
- 자동으로 다양한 크기 생성 (16x16, 32x32, 96x96)
- 적절한 `<link>` 태그 자동 삽입
- PWA manifest.json 자동 연동

#### 레거시 Favicon 아카이빙

**이동된 파일** (`/public/favicon/` → `/public/favicon-old/`):
- apple-touch-icon.png
- favicon-96x96.png
- favicon.ico
- favicon.svg
- site.webmanifest
- web-app-manifest-*.png

**제거된 불필요한 파일**:

## 🔧 Technical Improvements

### 1. Next.js 15 Favicon 통합

**layout.tsx 업데이트**:
```typescript
// src/app/layout.tsx

export const metadata: Metadata = {
  title: 'Mark25 - AI 상표 분석 서비스',
  description: '인공지능 기반 상표 분석 및 출원 대행 서비스',
  icons: {
    icon: '/favicon/icon.png',
    apple: '/favicon/apple-icon.png',
  },
  manifest: '/favicon/manifest.json',
}
```

**자동 생성 활용**:
- `icon.png` → 자동으로 favicon.ico 생성
- `apple-icon.png` → Apple touch icon 자동 처리
- 불필요한 수동 설정 제거

### 2. 성능 최적화

**이미지 최적화**:
- 로고 파일 크기 최적화 (66KB → 257KB, 고품질)
- 불필요한 placeholder 이미지 제거
- WebP 포맷 적극 활용 고려 (다음 릴리즈)

**번들 크기**:
- 레거시 favicon 파일 제거로 ~200KB 절감
- 최적화된 브랜드 자산으로 대체

### 3. 코드 품질 개선

**타입 안전성**:
- AdminService 타입 정의 개선
- 이메일 템플릿 타입 안정성 강화

**코드 정리**:
- 사용하지 않는 placeholder 제거
- 레거시 import 구문 정리
- 일관된 네이밍 컨벤션 적용

## 📚 Migration Guide

### For Production Deployment

#### 1. 환경 변수 확인
모든 환경 변수가 올바르게 설정되었는지 확인:
```bash
# 필수 확인 항목
RESEND_API_KEY=...           # 이메일 발송 (브랜드명 변경 반영)
NEXT_PUBLIC_APP_NAME=Mark25  # 새로운 앱 이름
```

#### 2. 빌드 및 배포
```bash
# 프로덕션 빌드
npm run build

# 보안 테스트 실행
npm run security:test

# 배포
npm run deploy
```

#### 3. DNS 및 도메인 설정
새로운 브랜드명이 반영된 도메인 확인:
```
기존: ipdoctor.com (또는 기존 도메인)
신규: mark25.com (새로운 도메인)
```

#### 4. 이메일 발송 도메인 검증
```bash
# Resend 대시보드에서 확인
- 발송자: Mark25 <noreply@mark25.com>
- SPF/DKIM 레코드 검증 완료 확인
```

### For Developers

#### 1. 로컬 환경 업데이트
```bash
# 최신 코드 pull
git checkout main
git pull origin main

# 의존성 업데이트
npm install

# 개발 서버 실행
npm run dev
```

#### 2. 브랜드 자산 경로 확인
새로운 경로로 이미지 참조:
```typescript
// 올바른 경로

<Image src="/logo-auth.png" alt="Mark25 Auth" />

// 제거된 경로 (사용 불가)

<Image src="/placeholder-logo.png" /> // ❌
```

#### 3. 이메일 템플릿 테스트
